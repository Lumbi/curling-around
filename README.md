# Curling Around

カーリングを基にしたゲームプロトタイプ

## 操作

| 入力          | アクション  |
| ------------ | ---------- |
| 左・右方向キー | 回転する   |
| 上方向キー    | 上から見る  |
| スペースキー   | 石を投げる  |

## パラメータ編集

`asset`フォルダーにパラメータ値を定義する`parameters.txt`があります。パラメータを編集するとほぼリアルタイムに実行中のゲームに反映されます。

`parameter-editor`のフォルダーにパラメータファイルを編集する簡易なプログラムのソースコードがあります。
